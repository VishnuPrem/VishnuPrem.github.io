Landmark-based Localisation of Indoor Mobile Robot

FEB 2018 - MAY 2018

Final year project done at Autonomous Systems and Advanced Robotics Research Centre, University of Salford, Manchester, UK


Source Code

Navigation, localisation and obstacle avoidance of Pioneer P3-DX mobile robot using RGB-D sensor wherein the robot identifies landmarks using retrained deep learning based object detection model. The robot navigates to different landmarks in the room and also localizes and predicts direction of hidden landmarks

Chess Playing Robot

NOV 2016 - MAR 2017

Third year group project presented at Annual Reseach Coloquim and unveiled at opening ceremony of Robotics Club at MAHE Dubai


Source Code

The chessbot is a robot that can play a complete game of chess against a human opponent. It uses image processing to track the movement of pieces by the human opponent. Then it calculates the best move to be made and picks and places the required piece to execute the best move. The process repeats till the game comes to an end. Programming using Python and OpenCV for image processing and Arduino Uno for motor control.

Motion following Camera

DEC 2016

Third year project for Microcontroller Based System Design Course at MAHE Dubai

A webcam captures images which are processed using Python with OpenCV. The direction of motion is determined by subtracting consecutive images to locate the area with the most non- black pixels in the resulting image. The Arduino controls a servo motor which is placed on the bottom of the webcam. Hence, the camera turns in the direction of the detected motion.
